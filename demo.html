Usage Examples / Примеры использования

Как использовать классы типографики:

<h1 class="heading-h1">Заголовок</h1>
<p class="body-01">Основной текст</p>
<span class="label">Метка</span>

Как использовать цвета:

<div class="bg-primary text-light-gray">
    Тёмный фон со светлым текстом
</div>

<button class="btn btn-primary">Кнопка</button>

CSS переменные:

var(--color-primary)
var(--color-accent)
var(--color-purple)
var(--font-family)
